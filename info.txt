    Основные команды:
    runserver
👉 python manage.py runserver
Запускает локальный сервер на http://127.0.0.1:8000/

startapp <имя>
👉 python manage.py startapp myapp
Создаёт новое приложение внутри проекта.

showmigrations
👉 python manage.py showmigrations
Показывает список всех миграций и их статус.

makemigrations
👉 python manage.py makemigrations
Фиксирует изменения в моделях (создаёт миграции).

migrate
👉 python manage.py migrate
Применяет миграции к базе данных.

createsuperuser
👉 python manage.py createsuperuser
Создаёт администратора для входа в админку.

shell
👉 python manage.py shell
Запускает интерактивный Python с доступом к Django-проекту.

check
👉 python manage.py check
Проверяет проект на ошибки конфигурации.

collectstatic
👉 python manage.py collectstatic
Собирает статические файлы в одну директорию (для продакшена).
